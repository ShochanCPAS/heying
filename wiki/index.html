    和瑛手册

    为了彰显我建筑社区特色，使得服务器更加美观，布局更加和谐。管理层特制定以下规定。

    玩家自建小镇相关细则

    • 服务器鼓励玩家自行成立建设聚落
    • 4人即可向管理申请登记建镇
    • 服务器公示已经成立的小镇，并注明活跃度，引导新玩家加入。/menu
    • 根据镇民意愿，管理可为玩家小镇设置传送点。/warp
    • 管理员有责任保护已登记的玩家聚落
    • 至少推举一名全镇代表或镇长，总负责小镇的建筑规划和小镇与管理之间沟通交流。
    • 建议小镇事先规划道路，居住范围等。
    • 建议小镇建设在平原地区。
    • 建议小镇核心地段圈好领地，以免破坏。
    • 务必要有规划地组织建设，表现规划之美，彰显建筑社区特色

    玩家基本权限细则

    Essentials基础插件使用说明

    经济类

    • 常用命令如下 /balance 查看金钱 /banlancetop 财富榜 /pay {玩家} {钱数} 给某人付款 /worth 查看某物品的价格 /sell 卖出手持物品

    功能类

    • 常用命令如下 /afk 进入挂机状态 /compass 查看你正对的方向 /depth 查看你的海拔 /getpos 查看坐标 /info 查看服务器信息 /list 查看在线玩家 /near 查看附近的玩家 /nick {昵称}设置昵称 /realname {昵称} 查看某昵称的真实id /recipe 查看如何合成一物品 /kit 领取礼包 /hat 把手持物品戴在头上 /fly 开关飞行（仅建筑认证玩家可享有）

    消息类

    • 常用命令如下 /ignore {玩家} 屏蔽一个玩家 /helpop 向op喊救命 /mail 发送邮件 /msg 私聊 /say 说话

    传送类

    • 常用命令如下 /spawn 传送到主城 /back 返回到上一个传送点 /sethome 设置家 /home 传送到一个家 /warp 传送到地标 /tpa 请求传送到一个玩家身边 /tpahere 请求一个玩家传送到自己身边 /tpyes 接受一个传送请求 /tpno 拒绝一个传送请求 /tptoggle 开关tp功能（关闭后无法发送和接受tp请求）

    玩家自建公会细则

    • 装设公会插件是为了满足团队游戏活动的需要。
    • 目前不开放公会战争、公会领地（可用res代替）

    Guild公会插件使用说明

    • 常用命令如下 /guild 获取全部命令 /guild info 公会主界面 /guild create {公会名} 创建一个公会，需花费1000游戏币 /guild invite {玩家id} 邀请一个玩家 /guild boot {玩家id} 踢出一个玩家 /guild accept 接受邀请 /guild decline 拒绝邀请 /guild request {公会名} 申请加入公会 /guild bank 公会银行相关 /guild buff 购买buff界面 /guild vault 公会存储仓库 /guild status 改变公会状态 /guild promote 更改会员等级权限 /guild chat 开启公会聊天频道

    玩家领地细则

    Residence领地插件使用说明

    • 使用木锄分别用左右键选取一个直平行六面体的两个对角，以圈定一个立体图形作为领地选区范围。
    • 常用命令如下 /res create {领地名} 选取后输入，创建属于你的领地 /res set {领地名} 打开领地权限设置菜单 /res pset {领地名} {玩家} 打开该玩家领地权限的设置菜单 /res message {领地名} {消息} 设置出入领地的消息 /res remove {领地名} 删除某领地 /res give {领地名} {玩家} 赠送某领地给某玩家 /res tpset 设置该领地的传送点 /res tp {领地名} 传送到某领地

    玩家储存容器保护细则

    • 被玩家上锁的箱子中的资源是无可非议的玩家的私有物品，服务器加以保护。
    • 对于没有被上锁的箱子，管理员会仅对在不同情形下的箱子中的重要物品进行人工保护。

    LWC箱子锁插件使用说明

    • 当你放置一个箱子或者任何能够储存物品的方块时，它默认的状态是 私有保护。
    • 私有保护：仅仅是放置者可以打开、拿取、放入和移除。
    • 公共保护：所有玩家均可打开、拿取、放入，但仅有放置者可以移除。
    • 捐赠保护：所有玩家均可打开、放入。但仅有放置者可以移除、拿取。
    • 密码保护：玩家只要输入密码就可以打开、拿取、放入。但仅有放置者可以移除。
    • 常用命令如下 /cremove 移除储存容器所带的所有保护 /cprivate 对该储存容器添加 私有保护 /cpubilc 对该储存容器添加 公共保护 /cdonation 对该储存容器添加 捐赠保护 /cpassword {密码} 对该储存容器添加 密码保护 /cunlock {密码} 解锁一个被密码保护的储存容器 /cinfo 查看该储存容器的保护状态

    玩家交易贸易细则

    • 服务器设置官方的资源买卖市场，以便玩家换取资源和兑换金钱。
    • 服务器鼓励玩家自建商店，进行交易贸易。
    • 玩家不通过Quickshop插件而私自进行的交易不受服务器保护。

    QuickShop插件使用说明

    • 常用指令如下： /qs create 对准箱子输入，创建一个正在手持的物品的商店。左键使用，右键设置，一看就会。

    建筑规范迁移制度

    • 对于建筑本身具有观赏价值，但是所处地不当，影响整体美观的建筑，建筑所有者可以向管理申请迁移。
    • 对于极为影响服务器美观的建筑，应当进行迁移。

    自然地形保护制度

    • 大规模的破坏地形是被禁止的（空置域、不合理平地等）
    • 使得地形不美观的行为是被禁止的
    • 保护地形连续性完整性。
    • 建筑规划主动与自然地形相适应。

    高能红石限制制度

    • 0tick高频红石是被禁止的。
    • 大规模的使用漏斗和红石粉是被禁止的。
    • 任何目的的高空流水是被禁止的。
    • 活塞飞行器是被禁止的。
    • 刷TNT是被禁止的。
    • 堆叠TNT矿车是被禁止的。

    其他便于玩家创意建筑的制度

    • 领地内开放飞行权限。
    • 通过建筑审核，获得建筑认证的玩家获得永久飞行权限（可开关）。
    • 若经济能力不足，具观赏价值的建筑可由管理代为圈地保护。
    • 即使没有圈地的具有观赏价值的建筑仍受管理保护。
    • 服务器建筑师优先原则。
    • 定期拍摄宣传片，宣传片中原创建筑的镜头注明建筑者。
    • 定期拍摄优秀原创建筑在官网进行展示，注明建筑者。
